# Notes

Instruments hold strike/maturity/type. Engines price them. MarketData holds spot, optional YieldCurve for discounting, optional VolSurface for strike vol.

## Engines

- AnalyticalEngine — European BS + greeks
- BinomialEngine — CRR, American early exercise
- MonteCarloEngine — GBM, antithetic and terminal-spot control variate
- FiniteDifferenceEngine — log-space PDE, explicit/implicit/CN

European: all four. American: binomial only. Barrier knock-out: FD. Asian arithmetic: MC.

AnalyticalEngine and MonteCarloEngine pull rate from `market.discount_factor(T)` and vol from `market.volatility(strike)` when curves are set.

## Conventions

Theta is per calendar day. Vega and rho are per 1% bump.

FD grid uses x = ln(S). MC control variate: discounted terminal spot (mean S0).

Benchmark numbers are in README; generated by `bench_pricing`.
